# classifies-students

This is a minimal React project skeleton created so you can connect to Supabase.

Prerequisites
- Node.js (LTS) and npm installed

Quick start
1. Install dependencies:

```powershell
npm install
```

2. Create a `.env.local` in the project root with your Supabase values (do not commit it):

```
REACT_APP_SUPABASE_URL=https://<your-project-ref>.supabase.co
REACT_APP_SUPABASE_ANON_KEY=<your-anon-public-key>
```

3. Start the dev server:

```powershell
npm start
```

Files created
- `package.json` - dependency manifest
- `public/index.html` - HTML entry
- `src/` - React source files including `App.js` and `supabaseClient.js`

Notes
- After installing Node/npm, run `npm install` to fetch dependencies.
- If you prefer I can later run the commands here once Node/npm are available.
